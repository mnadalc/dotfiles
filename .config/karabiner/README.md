# Karabiner Config

This folder tracks the Karabiner configuration used on this machine.

## Files

- `karabiner.json`: full Karabiner config (profile + complex rules).

## Recommended checks

- Karabiner -> Settings -> Virtual Keyboard -> `keyboard_type_v2`: use `iso`.
- Confirm complex modification rules are enabled.

## Current key remaps

Left Option (accents):

- `left_option + a` -> `á`
- `left_option + e` -> `é`
- `left_option + i` -> `í`
- `left_option + o` -> `ó`
- `left_option + u` -> `ú`
- `left_option + n` -> `ñ`
- `left_option + shift + a` -> `Á`
- `left_option + shift + e` -> `É`
- `left_option + shift + i` -> `Í`
- `left_option + shift + o` -> `Ó`
- `left_option + shift + u` -> `Ú`
- `left_option + shift + n` -> `Ñ`

Right Option (symbols/umlaut):

- `right_option + u` -> `ü`
- `right_option + shift + u` -> `Ü`
- `right_option + e` -> `€`
- `right_option + 3` -> `£`
